Project Overview

This web application allows users to manage and track their personal book collections efficiently. It includes features like automated ISBN search, user authentication, and role-based access to cater to different types of users, ensuring secure and personalized use.

Key Features

  • Automated ISBN search for quick and easy book tracking.
  • Secure user authentication with login and registration functionality.
  • Role-based access, allowing distinct permissions for admins and regular users.
  • Responsive and adaptive design, optimized for any device.

Technologies Used

  • Frontend: React.js, Bootstrap, HTML5, CSS3
  • Backend: Node.js, PHP, MySQL
  • API: RESTful API
  • Authentication: JSON Web Token (JWT)
