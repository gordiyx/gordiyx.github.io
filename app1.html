Project Overview

This web application allows users to manage and track their personal book collections efficiently. It includes features like automated ISBN search, user authentication, and role-based access for different user types.

Key Features

  • Automated ISBN search for quick and easy book tracking.
  • Secure login and registration functionality.
  • Role-based access for different user permissions.
  • Responsive design optimized for any device.

Technologies Used

  • Frontend: React.js, Bootstrap, HTML5
  • Backend: Node.js, PHP, MySQL
  • API: RESTful API
  • Authentication: JWT (JSON Web Token)
